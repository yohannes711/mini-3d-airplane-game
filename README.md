# mini-3d-airplane-game
Game web 3D menggunakan Three.js di mana pemain mengendalikan pesawat untuk mengumpulkan energi dan menghindari rintangan

## 🛠️ Teknologi yang Digunakan
- **HTML5 & CSS3** (Struktur UI & Desain Animasi)
- **JavaScript (ES6)** (Logika Game & Interaksi)
- **Three.js** (Rendering Grafis dan Objek 3D)
- **TweenMax (GreenSock)** (Halus animasi pergerakan objek)

## 🎮 Cara Bermain
1. Klik kiri pada mouse/layar untuk memulai game dan memunculkan suara latar.
2. Gerakkan mouse ke atas/bawah untuk mengendalikan posisi terbang pesawat.
3. **Misi:** Ambil batu **Biru** untuk mengisi energi dan hindari batu **Merah** agar tidak meledak!
